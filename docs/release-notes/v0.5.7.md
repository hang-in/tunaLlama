### 새 CLI - `tunallama metrics`

```bash
tunallama metrics show              # metric 별 avg / count / min / max
tunallama metrics show --source organic   # source 필터
tunallama metrics list              # 최근 50 entry list
tunallama metrics path              # db 파일 경로
tunallama metrics clear             # 전부 삭제
tunallama metrics clear --source organic   # source 필터 삭제
```

### 새 env vars

- `TUNA_ORGANIC_METRICS` (default `1`): 자동 수집 on/off
- `TUNA_METRICS_DB`: db path override (default `~/.tunallama/metrics.db`)
- `TUNA_STATE_BASE` (이미 v0.5.2): state.md base override
