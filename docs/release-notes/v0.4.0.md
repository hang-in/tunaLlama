## 새 mechanism (Phase 6-1 ~ 6-4)

### 6-1. Project-scoped state.md auto-load

`~/.tunallama/projects/<sha256-12hex>/state.md`. git root 자동 감지.

```markdown
# tunaLlama Project Memory

## Conventions
- (manual) use `MemoryStore` not `Store`

## Active Decisions
- (auto, 3 occurrences) HyDE hybrid 채택 - production winner on synthetic seed (P@1 0.92)

## Constraints
- (manual) backend 은 plugin 을 import 하지 않음

## Anti-patterns observed
- (auto, 9 occurrences) standalone-toy 패턴 (round 7-15)
```

Auto-load 경로:
1. **MCP resource** (`tunallama://memory/state`) - Claude Code 자동 attach
2. **Tool fallback** (`tuna_load_memory`) - 명시 호출

Size cap 2KB. `Conventions` / `Constraints` 는 never truncate. `Active
Decisions` → `Anti-patterns` 순 truncate.

### 6-2. Decision / convention / constraint / antipattern 자동 추출

`tunallama_core/memory/extract.py`. regex + rule-based.

매 delegation 직후 (`tuna_dev_review`, `tuna_general_task` 등) state.md 에
자동 append. case-insensitive text dedup + optional BGE-M3 embedding dedup
(cosine ≥ 0.85).

env `TUNA_AUTO_EXTRACT_STATE=0` 으로 비활성 가능. 실패해도 silent
(delegation 자체에 영향 X).

### 6-3. Diff-based learning

`calls.target_file_path TEXT` 컬럼 추가 (idempotent migration).

```python
# 두 방법:
store.record_call(..., target_file_path="/path/to/file.py")  # 기록 시
store.set_target_file_path(call_id, "/path/to/file.py")       # 사후 link
```

`tunallama_core/memory/diff_learn.py`:
- rule-based: `Store` → `MemoryStore` 같은 identifier rename / import
  path 변경 자동 감지.
- LLM-based fallback: 복잡한 rewrite 는 옵션 LLM 호출. **state.md
  미주입 모델 권장** (dependency loop 회피).

### 6-4. 자동화 metrics

`tunallama_core/measurement/memory_metrics.py`. 모든 sample 에 `source`
tag (`synthetic` / `spec_dogfooding` / `organic`).

| metric | 정의 |
|---|---|
| convention_adherence_rate | state.md 의 각 convention 이 output 에서 honor 된 비율 |
| standalone_toy_rate | Mock / np.random / SyntaxError / unrelated keyword 출현 비율 |
| user_intervention_rate | target_file 의 LLM 출력 대비 사용자 수정 severity 평균 |
| state_recall_rate | state.md 의 convention 에 대한 모델 응답 일치 비율 |

**중요**: 절대 threshold 미리 X. trend over time + source 분리 트래킹.
