### SessionStart hook 추가

`plugin/hooks/session_start.py` (옵션 메커니즘):

- 세션 시작 시 자동 실행 (클라이언트 지원 시)
- `state.md` 내용을 stdout 으로 JSON 출력 (`additionalContext` field)
- 빈 state 면 silent (가치 X 시점)
- `tunallama_core` 못 찾는 환경에서도 silent 종료 (다른 환경 안 막음)

`plugin/hooks/hooks.json` (신규):

```json
{
  "hooks": {
    "SessionStart": [
      {"hooks": [{"type": "command", "command": "python ${CLAUDE_PLUGIN_ROOT}/hooks/session_start.py"}]}
    ],
    "PreToolUse": [
      {"matcher": "Read", "hooks": [{"type": "command", "command": "python ${CLAUDE_PLUGIN_ROOT}/hooks/pre_tool_use.py"}]}
    ]
  }
}
```

플러그인 설치 시 자동 등록 (클라이언트 hooks.json 지원 시). 미지원 시
사용자가 `~/.claude/settings.json` 등에 수동 등록 가능.

### `TUNA_STATE_BASE` env 추가

테스트 / multi-tenant 환경에서 state.md base path override 가능. 기본은
그대로 `~/.tunallama/projects`.
